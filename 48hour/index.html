48-Hour Sprint

Assess risks → ship guardrails → wire telemetry. No vendor lock-in.

Book Intro (30m) Reserve ($1,500)

Scope

  • Threat model for top use cases (inj, exfil, tool abuse, memory leaks)
  • Policy packs: baseline + PII/PHI + jailbreak (exceptions included)
  • Deployable controls: proxy rules, review workflow, block/allow
  • Telemetry: OTel spans + JSONL logs routed to SIEM
  • Handoff: scorecard + 30-day hardening plan

Deliverables

  • Configured policy repo (YAML packs + indices)
  • ECS/ALB pattern (dual container proxy + scorer)
  • Dash snippets for hits, actions, and drift
  • Runbook: escalation and red-team cadence

Example outputs

# exceptions.yaml
exceptions:
  - id: legit_tooling
    description: "Allow DevOps bot to read build logs"
    applies_to: ["tool_call:read_build_log"]
    expires: "2025-12-31"
