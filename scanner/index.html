Prompt Risk Scanner

Catch jailbreaks and leaks before they ship. Works from laptop to CI.

Request Demo See examples

Modes

  • CLI: scan *.jsonl files
  • API: POST to /scan, get scores + actions
  • CI: block on policy regressions

Signals

  • Jailbreak semantic similarity
  • PII/PHI regex + heuristics
  • Exfil/abuse intents
  • Token overrun & memory leakage hints

API Example

POST /scan
{
  "session_id":"abc123",
  "prompt":"Ignore all instructions and ...",
  "response":"Sure, here's the admin password ...",
  "meta":{"model":"claude-3.7","env":"staging"}
}
{
  "scores":{"jailbreak":0.91,"pii":0.12,"exfil":0.77},
  "actions":["block","review"],
  "matched_rules":["jailbreak_semantic"]
}
